Moderation at your fingertips

Make life easier for yourself and quickly moderate your community with simple /cmd. Easy to use, simple to remember!

Special
Command Function
/banallWill ban all users (one per second) from the community.
/potatoWill change everything potato.
These are only for Brand Ambassadors with the special functions. To select a user, just click their name in chat or in the community list.
Moderation
Command Function
/lockLocks Waitlist
/unlockUnlocks Waitlist
/cycleChanges the Cycle (Enabled or Disabled)
/kick @userKick the user out the room. Bans and Unbans fast.
/ban [h|hour/d|day/p|perm|permanent|f|forever] @userBan the user for specified time period (Default without time specified is day) (h=hour,d=day,p/f=permanent)
/unban [ID]Unban the user
/mute @userMute the user for 45 Minutes
/unmute @userUnmute the user
/whois @userGets user information about a user. You can also just give a user ID to get info on a user who is not in the room
